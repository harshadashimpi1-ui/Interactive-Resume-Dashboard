📄 Interactive Resume Dashboard

A modern, interactive resume built in Power BI that transforms a traditional CV into a dynamic portfolio. The dashboard enables recruiters to explore projects, technical skills, certifications, education, achievements, and contact information through an intuitive and visually engaging interface.

📌 Short Description / Purpose

The Interactive Resume Dashboard is designed to present professional qualifications in a more engaging and interactive format than a conventional resume. Built using Power BI, it provides easy navigation across different sections while showcasing projects, technical skills, certifications, education, and achievements in a single dashboard.

🛠️ Tech Stack

The dashboard was built using the following tools and technologies:

📊 Power BI Desktop – Dashboard development and interactive report creation.
🔄 Power Query – Data preparation and transformation.
🧮 DAX (Data Analysis Expressions) – Measures, calculated fields, and KPIs.
🗂️ Data Modeling – Organizing resume information into related tables.
📁 Power BI Service – Dashboard publishing and online sharing.
📂 Data Source

All data used in this dashboard was created and maintained manually to represent my professional profile.

The dataset includes:

Personal Information
Education
Technical Skills
Projects
Certifications
Achievements
Contact Information
Career Objective

Note: This dashboard uses self-created resume data and does not rely on any external dataset.
